2018 旅行日记 睢县

发表于 2018-11-07 | 更新于 2018-11-11 | | 阅读次数：

旅行日记 2018年暑假，我和朋友出去玩，拍了一些照片。

每个少年都好像一只雏翼的鸟，当蒙在生命里的黑布被掀起一角，露出一抹剌眼的天光——青春，于是被惊醒了

位置境域:
睢县是商丘市下辖县，位于华北平原南端、豫东平原东部，西距省会郑州市160公里，东距京九、陇海两大铁路交汇处商丘市区60公里，东邻宁陵县，南接柘城县、太康县，西毗杞县，北靠民权县，东西宽约33公里，南北长约41公里，总面积926平方公里，其中耕地面积100.3万亩。 –百度百科
