Introduction

DPlayer is a lovely HTML5 danmaku video player to help people build video and danmaku easily.

DPlayer supports:

  • Streaming formats
    • HLS
    • FLV
    • MPEG DASH
    • WebTorrent
    • Any other custom streaming formats
  • Media formats
    • MP4 H.264
    • WebM
    • Ogg Theora Vorbis
  • Features
    • Danmaku
    • Screenshot
    • Hotkeys
    • Quality switching
    • Thumbnails
    • Subtitle

Using DPlayer on your project? Let me know!
